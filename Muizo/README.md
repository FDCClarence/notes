# Muizo

Minimal React + Vite app with Tailwind CSS v3.

## Setup

```bash
npm install
npm run dev
```

## Structure

```
src/
  components/   # UI components
  hooks/        # custom React hooks
  config/       # app configuration
  assets/       # images, fonts, etc.
```

## Stack

- React 19
- Vite 8
- Tailwind CSS v3
- @tailwindcss/typography
